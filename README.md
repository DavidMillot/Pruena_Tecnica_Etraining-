# Pruena_Tecnica_Etraining-
Proyecto de análisis de datos que integra información de ventas y clima mediante un proceso ETL, realiza análisis descriptivo y un modelo de regresión lineal para evaluar la relación entre la precipitación diaria y el volumen de ventas.

## Estructura del repositorio
- **notebook/**
  - Notebook principal con el flujo completo de ETL, análisis y conclusiones.
- **data/**
  - Dataset final limpio en formato Excel.
- **powerbi/**
  - Archivo Power BI con la visualización de resultados.

## Ejecución

El notebook contiene el desarrollo completo del análisis y puede ejecutarse en Google Colab o Jupyter Notebook.
